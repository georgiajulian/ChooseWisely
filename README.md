# 🛡️ ChooseWisely: Interactive AI Anti-Scam Drama

**ChooseWisely** is an interactive, AI-generated cinematic experience designed to teach users how to spot and avoid modern scams. Users step into a realistic, branching narrative where their choices directly shape the story, visuals, and outcome.

---

## 🌟 Key Features

- 🎬 **Cinematic AI Video**: Uses Alibaba Cloud Wan 2.7 to generate unique, 5-second cinematic clips tailored to user choices.
- 🎭 **Dynamic Character Voices**: Features distinct AI voices (Scammer, Victim, Narrator, Police, Bank) with automatic text-cleaning to remove awkward speaker labels and emojis.
- 🎧 **Dynamic Audio Design**: Layers suspenseful background drones during buildup, and triggers distinct "Win" or "Defeat" sound effects based on the outcome.
- 🧠 **State-Aware Memory**: Tracks red flags, money lost, and danger levels to ensure narrative consistency across multiple scenes.
- 🎞️ **Movie Compilation**: Stitches all scenes together into a single, seamless MP4 file for the user to download.
